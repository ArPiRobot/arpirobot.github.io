Detailed Description

Drive helper for tank drive method.

Drive helpers calculate speeds for each motor based on a set of inputs and update all motors speeds at once. Tank drive uses two inputs: left speed and right speed.

Set a new left and right speed value at the same time.

Speeds will be re-calculated and updated for each motor. If updating both values use this instead of calling updateLeftSpeed and updateRightSpeed because this will only recalculate motor speeds once.

Parameters
new_left_speedthe new speed (-1.0 to 1.0)
