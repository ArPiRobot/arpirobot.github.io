Detailed Description

Drive helper for arcade drive method.

Drive helpers calculate speeds for each motor based on a set of inputs and update all motors speeds at once. Arcade drive uses two inputs: speed and rotation.

Set a new speed and rotation value at the same time.

Speeds will be re-calculated and updated for each motor. If updating both values use this instead of calling updateSpeed and updateRotation because this will only recalculate motor speeds once.
